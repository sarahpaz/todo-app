# Todo App - Vue.js

Todo App build from [Traversy Media Vue JS Crash Course](https://youtu.be/Wy9q22isx3U)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
